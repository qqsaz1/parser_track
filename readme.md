# Инструмент для очистки файлов треков в формате GeoJSON.

## Описание

Скрипт обрабатывает файлы геоданных (включая текстовые файлы с GeoJSON-структурой типа FeatureCollection), выполняя две основные задачи:

1. **Очистка данных**: Из структуры трека полностью удаляются все свойства и избыточные метаданные (скорость, заглубление орудия, телеметрия, передачи). В массиве координат сохраняются исключительно пары значений широты и долготы `[longitude, latitude]`.
2. **Фиксированный шаг (3 метра)**: Алгоритм выполняет интерполяцию линии маршрута (`LineString`). Независимо от частоты исходных точек, расстояния между ними или характера движения (прямые участки, повороты, развороты), результирующий трек перестраивается с фиксированным шагом ровно 3 метра между соседними точками.

## Требования

- Python 3.x
- Встроенные модули `json` и `math` (установка дополнительных сторонних библиотек не требуется).

## Структура обрабатываемых данных

Скрипт рассчитан на стандартный формат GeoJSON (с расширением `.geojson`, `.json` или `.txt`), содержащий объект типа `FeatureCollection` с геометрией `LineString` внутри массива `features`.

## Использование

1. Поместите исходный файл трека в `track/`папку со скриптом.
2. В блоке `if __name__ == "__main__":` укажите имена файлов:
   ```python
   INPUT_FILE = "track/имя_исходного_файла.txt"
   OUTPUT_FILE = "имя_выходного_файла.txt"
   ```

## Запуск

```bash
python main.py
```
